# Machine Learning — Travaux pratiques

Dépôt des travaux du cours d'Intelligence Artificielle (2ème année, 2025-2026).  
**Auteurs :** Adrien Dai Pra & Victor Van Zieleghem

## Structure du projet

| Dossier | Contenu |
|---------|---------|
| `partie1_titanic_analysis/` | Analyse exploratoire du dataset Titanic (EDA, statistiques, visualisations) |
| `partie2_titanic_preprocessing_modeling/` | Prétraitement, feature selection et modélisation (KNN, Decision Tree, Random Forest) |
| `partie3_titanic_pipeline/` | Pipeline complet prétraitement + modèle en un seul notebook |
| `partie4_vectordb_embeddings/` | Base de données vectorielle FAISS + classification d'articles par embeddings |
| `partie5_agent_ia/` | Agent IA (outils, tests de comportement) |
| `_misc/` | Fichiers divers / temporaires |
